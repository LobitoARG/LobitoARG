Right now my main proyect is: [Argentium3.0 E-Commerce](https://github.com/LobitoARG/Grupo_3_Argentium3.0)

The webpage allows the user to complete a register form, log in wich is connected to a local database
Each form has his own validatation using middlewares and js 

Also, you can simulate buying an item by adding it to the cart. All the items are saved using Local Storage.

The webpage is fully responsive and can adapt to any resolution.

I also developed a dashboard where the Admin can see all the products, registered users and the idea is to move all the CRUD functions to this app wich was created using ReactJs.
By using Browser Router/Routes/Route/NavLink i was abel to create a SPA webpage.
